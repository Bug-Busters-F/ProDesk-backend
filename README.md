# ProDesk - Backend

Este é o repositório do backend da aplicação **ProDesk**, um projeto desenvolvido pela equipe Bug Busters para o **5º semestre de Análise e Desenvolvimento de Sistemas** da Fatec São José dos Campos.

O ProDesk é um sistema de **Help Desk para serviços de TI**, desenvolvido para permitir o registro, acompanhamento e gerenciamento de chamados técnicos. A plataforma busca centralizar as solicitações de suporte, facilitando a comunicação entre usuários e equipe de TI e melhorando a organização do atendimento.

Acesse o repositório principal do projeto aqui ->  
<a href="https://github.com/Bug-Busters-F/ProDesk">ProDesk</a>

Instruções para rodar o projeto acesse  
<a href="./CONTRIBUTING.md">CONTRIBUTING.MD</a>

Este backend é responsável por:

- Servir a API que se comunica com a aplicação mobile.
- Gerenciar usuários e autenticação.
- Registrar e atualizar chamados de suporte.
- Armazenar e gerenciar os dados do sistema.

O objetivo é fornecer uma API organizada e eficiente para garantir o funcionamento do sistema de atendimento técnico.
